Model Comparison
April 2025
Palina Pyachynskaya

This Python program takes in a CSV of data and utilizes various ML models, including the multinomial Naive Bayesian, K Nearest Neighbors, and Random Forest models to predict new data. All models used are first trained on the provided data.
Various metrics, such as accuracy and precision, are then used to compare and analyze the performance of each model.

Used for CS 4341 Introduction to Artificial Intelligence.
